Object and Wall Following Robot Implementation for the Romi Car chassis. Built for CECS 347 Embedded Systems Class in CSULB.

Pinouts:
Romi Car Chassis:
 Left Motor
 PWM = PB4
 DIR = PB2
 SLP PB3
 
 Right Motor:
 PWM = PB5
 DIR = PB6
 SLP = PB7

IR Sensors:
PE0 = Left IR
PE1 = Front IR
PE2 = Right IR
